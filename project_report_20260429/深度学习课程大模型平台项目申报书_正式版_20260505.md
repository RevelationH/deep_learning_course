# 深度学习课程大模型平台项目申报书

## 一、项目概述

本项目面向课程大模型教学场景，围绕深度学习课程构建了一套“教学资料整理、课程知识库构建、智能问答、在线练习、学习报告分析”一体化平台。平台已形成课程资料、知识点、题库、问答服务、学习报告与历史记录等完整能力模块，能够面向课程教学提供统一的知识服务入口。

- 原始教学单元数量：1201
- 知识点数量：13

- Dense 检索单元数量：1206
- 向量模型：sentence-transformers/all-MiniLM-L6-v2

## 三、知识点清单

- 深度学习概述（Week1）：理解深度学习的研究目标、发展背景、表示学习思想，以及其与传统机器学习方法的联系。
- 数学基础与机器学习基础（Week2）：掌握深度学习所需的线性代数、概率统计、优化和监督学习基本概念。
- 神经元、感知器与反向传播（Week2、Week3）：理解人工神经元、感知器、多层感知器以及反向传播算法的核心机制。
- 卷积神经网络基础（Week3、Week4）：掌握卷积层、池化层、感受野、参数共享等 CNN 基本结构与核心思想。
- 典型卷积网络与视觉应用（Week4、Week5）：了解 LeNet、AlexNet、VGG、ResNet 等典型 CNN 架构及其在视觉任务中的应用。
- 循环神经网络与LSTM（Week6）：理解 RNN 的序列建模能力、梯度传播问题，以及 LSTM 等改进结构的作用。
- 注意力机制与Transformer（Week7）：掌握注意力机制、自注意力、位置编码及 Transformer 编码器-解码器结构。
- GPT、BERT与视觉Transformer（Week8）：理解基于 Transformer 的预训练模型家族，包括 GPT、BERT 与 Swin Transformer 的差异。
- GAN 对抗生成（Week8、Week9）：理解生成器与判别器的对抗训练机制，以及条件 GAN、结构优化等扩展思路。
- 深度生成模型与扩散模型（Week10）：掌握自编码器、RBM、DBN、Hopfield 网络及扩散模型等深度生成模型的核心思想。
- 正则化与优化（Week11、Week12）：理解参数范数正则化、数据增强、提前终止、Dropout、归一化及优化方法对泛化的影响。
- 深度学习开发平台（Week13）：熟悉 TensorFlow、PyTorch、Keras、PaddlePaddle 等开发平台及其特点。
- 决策树、随机森林与 DeepForest（Week11）：理解树模型、随机森林与 DeepForest 的集成学习思想及其与深度网络的关系。
